⚡ Core Concepts
Neuron — receives inputs × weights + bias → activation function → output.

Backpropagation — error propagates backward; weights updated via gradient descent.

Activation Fns — ReLU, Sigmoid, Tanh, Softmax, GELU.

Dropout — randomly disables neurons during training to prevent overfitting.

Core ML Algorithms
The essential toolkit — what each does and when to use it
Regression
Linear Regression
Predicts continuous values. Fits best line minimizing MSE. Use: price prediction, sales forecasting.
Classification
Logistic Regression
Binary classification via sigmoid. Outputs probability 0–1. Use: spam, churn, medical yes/no.
Tree-Based
Decision Tree
Splits on feature thresholds. Highly interpretable. Prone to overfitting — use ensembles instead.
Ensemble
Random Forest
Bags many decorrelated trees, averages predictions. Robust, handles missing values well.
Ensemble
XGBoost / LightGBM
Boosting: sequentially corrects errors. Dominates tabular ML competitions. Fast, regularized.
Instance-Based
K-Nearest Neighbors
Classifies by majority vote of k nearest neighbors. No training phase, lazy learner.
Kernel
Support Vector Machine
Finds max-margin hyperplane. Kernel trick handles non-linear boundaries. Good for high dimensions.
Clustering
K-Means / DBSCAN
K-Means: assign to nearest centroid. DBSCAN: density-based, finds arbitrary shapes, handles noise.
Probabilistic
Naive Bayes
Bayes' theorem + feature independence assumption. Lightning-fast, great for text/NLP classification.
Dimensionality
PCA / t-SNE / UMAP
PCA: linear compression. t-SNE/UMAP: non-linear visualization. Reduce features, remove noise.
Deep Learning
Transformer (Attention)
Self-attention on full sequences. Scales to billions of parameters. Foundation of all modern LLMs.
Reinforcement
Q-Learning / PPO
Agent maximizes cumulative reward. PPO = stable policy gradient used in ChatGPT's RLHF training.

Core LLM Concepts
🪙 Token
The unit of text LLMs process — roughly a word or subword (~4 chars). "ChatGPT" = 1 token. "tokenization" = 3 tokens. Models have a max token limit (context window).
📐 Context Window
Maximum tokens the model can "see" at once — its working memory. GPT-4: 128K. Claude 3.5: 200K. Longer = more expensive but handles long docs, code, conversations.
🌡️ Temperature
Controls randomness. 0 = deterministic (always picks most likely token). 1+ = more creative/random. Use 0 for factual tasks, 0.7–1 for creative writing.
🔢 Embedding
A dense vector (list of numbers) representing text, images, or any data in a high-dimensional space. Similar meanings → nearby vectors. Powers semantic search and RAG.
🔝 Top-P / Top-K
Sampling strategies. Top-K: consider only k most likely tokens. Top-P (nucleus): consider tokens whose cumulative probability ≥ P. Controls diversity of outputs.
🎯 Prompt Engineering
Crafting inputs to steer model behavior. Includes zero-shot (no examples), few-shot (give examples), chain-of-thought (ask to reason step-by-step), system prompts.
👻 Hallucination
When an LLM confidently generates false or fabricated information. A fundamental limitation. Mitigated by RAG, grounding, and verification pipelines.
🧩 System Prompt
A hidden instruction given to the LLM before the user message. Sets persona, tone, rules, and context. Powers products like Claude, ChatGPT, Copilot.
⚡ Inference
Using a trained model to generate outputs on new inputs. As opposed to training. Inference is what happens every time you chat with an AI — much cheaper than training.
🔧 Fine-tuning
Further training a pre-trained model on a smaller, task-specific dataset. Cheaper than training from scratch. LoRA and QLoRA are popular efficient fine-tuning methods.
🗄️ Vector Database
Stores embeddings for fast similarity search. Core component of RAG. Examples: Pinecone, Weaviate, Chroma, pgvector. Enables semantic search at scale.
🤖 AI Agent
An LLM with tools and memory that autonomously plans and executes multi-step tasks. Can call APIs, run code, browse web. Foundation of Agentic AI workflows.
🚀 GenAI (Generative AI)
AI systems capable of generating novel text, images, or other media in response to prompts, driving the latest wave of business transformation.
📚 RAG (Retrieval-Augmented Generation)
A technique where an AI model retrieves facts from an external database (like your company documents) before answering, ensuring high accuracy.
🤖 Copilot
An AI assistant integrated directly into applications (like Word, coding software, or internal tools) to augment human productivity.
🛡️ Guardrails
Safety mechanisms and policies applied to AI outputs to prevent brand damage, bias, or data leakage in enterprise environments.
🏗️ Foundation Model
Massive, generalized AI models (like GPT-4 or Claude) trained on vast data, which can be adapted or fine-tuned for specific business tasks.
Model Evaluation Terms
📉 Perplexity
Measures how well an LLM predicts a text sample. Lower = better. Key metric for language model evaluation.
📊 BLEU / ROUGE
BLEU: measures generated text vs reference (used in translation). ROUGE: measures recall, used in summarization evaluation.
🔄 RLHF
Reinforcement Learning from Human Feedback. Humans rank outputs; reward model trains on that; PPO fine-tunes LLM. How ChatGPT/Claude became helpful.
⚖️ Constitutional AI
Anthropic's technique (used in Claude): model critiques and revises its own outputs against a set of principles. Reduces need for human labeling.
